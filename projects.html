NBSC GoEnroll

UV-MICOSTEDI Box: Ultra-Violet Micro-Controlled Sterilizing and Disinfecting box

MICOSTEDI Box is an Arduino-powered UV sterilizing and disinfecting box designed to kill bacteria on everyday items. It uses a UVC lamp and heat from incandescent bulbs to eliminate harmful microorganisms within minutes. The system automates the process through programmed time settings that control exposure duration. MICOSTEDIOX offers a low-cost, chemical-free solution for safe sanitation of gadgets, bills, and small objects.

TaskTrek: Java-Based To-Do List Application

TaskTrek is a Java-based to-do list application designed to help users manage daily tasks efficiently. It features a simple graphical interface built using Swing. Users can create, edit, and delete tasks, set deadlines, and mark tasks as completed. Each task can include a title, description, and due date for better organization. The system uses file handling to store user data, ensuring that tasks are saved and retrieved even after closing the program. It also supports user authentication, allowing each user to manage their own to-do list securely. TaskTrek applies object-oriented principles and event-driven programming to deliver an interactive and responsive experience. The goal is to help users stay organized, meet deadlines, and improve productivity through a lightweight, easy-to-use task management tool
